# Mike's Auto Shop — Shop Management System

A dark-themed React dashboard for managing auto repair tickets, mechanics, and shop workflow.

## Setup

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`.

## Build for Production

```bash
npm run build
```

Output goes to `dist/`.

## Stack

- React 18
- Vite
- Outfit + JetBrains Mono fonts (loaded via Google Fonts)
